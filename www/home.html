Uniweb API Client Dashboard

This dashboard offers a list of links to API queries that demonstrate common operations. The source code of the queries can be found at the queries sub-folder. Please read the individual example source code to know more about each query.

Example API queries

Query Description
Fetch Profile data Generate an HTML page using JSON data fetched by the API client.
Fetch CV data Request information about sections, fields * and the valid options for drop-down fields.
Add items to Profile Add two research interests to the profile of a user. Note: Requires API write mode.
Add items to CV Add new items into a CV section. Note: Requires API write mode.
